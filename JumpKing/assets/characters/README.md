# Character Art

Runtime character sprites live here.

Built-in characters:

- `fighter/`
- `samurai/`
- `shinobi/`

Custom character override:

- `custom/player.png`

Recommended runtime file:

- `custom/player.png`

You can use either:

- A single transparent sprite image.
- A simple horizontal spritesheet.

If you use a 4-frame strip, the game reads it as:

1. idle
2. walk
3. charge
4. airborne

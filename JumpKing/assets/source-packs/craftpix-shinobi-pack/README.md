# Craftpix Shinobi Pack Archive

This folder is the archived source bundle for the imported character pack.

## Layout

- `sprites/`: original per-character exported PNG strips
- `source-sheets/`: original PSD spritelists
- `docs/`: license, coupon, URL, and pack metadata files

Runtime game assets do not load from here anymore.
They now live in `assets/characters` and `assets/models`.
